환율우대 서비스 안내

고객님이 외화를 보다 유리한 조건으로 환전하실 수 있도록, 플로은행은 온라인·모바일·영업점 채널별 환율 우대 서비스를 운영하고 있습니다. 거래 실적, 고객 등급, 상품 가입 여부에 따라 추가 혜택이 자동으로 적용됩니다.

1. 우대율 기준

  • 온라인·모바일 환전: 주요 통화 기준 최대 90% 우대 (USD, EUR, JPY 등)
  • 영업점 환전: 최대 50% 우대 (지점·고객 등급별 차등)
  • 공항환전소: 사전예약 고객 대상 30~70% 우대율 적용

2. 우대 신청 방법

  • 모바일 앱 또는 인터넷뱅킹 접속 → ‘환전우대 신청’ 메뉴 선택
  • 환전 금액과 통화를 입력하면 자동으로 우대율이 적용되어 표시
  • 적용 내역은 환전 영수증 및 거래내역에서 확인 가능

3. 추가 혜택

  • 외화예금 고객: 환전 시 기본 10% 우대 자동 적용
  • 급여이체 고객: USD·JPY·EUR 환전 시 추가 5% 우대
  • 해외송금 연계: 연 2회 송금 수수료 면제

4. 유의사항

  • 환율우대는 시장 환율, 거래 시점, 지점별 정책에 따라 변동될 수 있습니다.
  • 일부 통화(중남미·아프리카 등)는 우대율 적용이 제한됩니다.

※ 환율우대율은 신청 시점 기준이며, 거래 변경 시 우대율이 소멸될 수 있습니다.
※ 실제 적용 환율은 최종 결제 시점의 고시 환율 기준으로 계산됩니다.
